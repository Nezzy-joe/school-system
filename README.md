# School Management System Backend

A backend system for managing school operations, built with **Golang**, **PostgreSQL**, and **JWT authentication**.

This project is being developed step-by-step as a real-world school management platform. The goal is to support student records, subjects, results, fees, reports, broadsheets, and future frontend dashboard integration.

---

## 🚀 Features Completed

- User registration
- Secure password hashing with bcrypt
- User login
- JWT token generation
- Protected routes using middleware
- Student creation
- Fetch students
- Subject creation
- Fetch subjects
- PostgreSQL database integration

---

## 🏫 Planned Features

- Result management
- Grade calculation
- Termly report generation
- Broadsheet generation
- Excel score upload
- GPA/CGPA toggle
- Fees management
- Admin dashboard frontend
- Role-based access control for admin, teachers, and students

---

## 🛠️ Tech Stack
